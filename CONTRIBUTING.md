# Contributing to Stellar RWA Guard

We are participating in the **Stellar Wave Program**! We welcome contributors of all skill levels.

### How to get started:
1. Look for issues labeled `Stellar Wave`.
2. Comment on the issue to get assigned.
3. Submit a Pull Request (PR) with your changes.

### Rewards:
By contributing code, documentation, or tests, you earn **Points** on Drips which are converted into rewards at the end of the Wave cycle.

### Code Standards:
- All smart contracts must be written in Rust (Soroban SDK).
- Please include comments explaining your logic.
