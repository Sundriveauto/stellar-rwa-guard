# Stellar RWA Guard 🛡️

Stellar RWA Guard is an open-source framework built on **Soroban** to help asset issuers automate compliance for Real-World Assets (RWAs). 

### Why this exists:
As the Stellar network scales with institutional assets, issuers need a standardized way to manage KYC/AML whitelists on-chain without building from scratch.

### Current Features:
- ✅ On-chain Whitelist Storage
- ✅ Admin-only Authorization Logic
- ✅ Lightweight Soroban Contract
